## 소개
- 최신 유행하는 영화를 알아보아요
- 원하는 영화를 검색하고 찾아볼 수 있어요
- 필터를 이용해서 원하는 장르의 영화를 찾아볼 수 있어요
- 맘에 드는 영화는 DB에 저장하고 나중에 확인할 수 있어요
